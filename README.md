## Overview

Marketplace landing page for connecting companies with vetted senior professionals. Rebuilt from a legacy Next.js 9 app into a modern **React + TypeScript + Vite + Tailwind v4** stack with shadcn-style components.

Demonstrates product landing patterns: hero, feature cards, professional directory, and clear CTAs.

---

## Features

- Hero with dual CTAs (Find talent / Browse projects)
- Feature highlights — Vetted network, Fast matching, Enterprise ready
- Featured professionals grid with rates
- Dark slate design system
- Fully responsive

---

## Tech Stack

React 19 · TypeScript · Vite · Tailwind CSS v4 · lucide-react · CVA

## Quick Start

```bash
git clone https://github.com/jonathasribeiro/ineed-inc.git
cd ineed-inc
npm install
npm run dev
```

### Docker

```bash
docker build -t ineed-inc .
docker run -p 8080:80 ineed-inc
```
